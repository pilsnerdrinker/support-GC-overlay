# GCカンペ デスクトップオーバーレイ / GC Cheat Sheet Desktop Overlay

FF14のP4グランドクロス用カンペを、デスクトップ上に半透明の小さい画面として表示するElectron版です。

This is an Electron desktop overlay for the P4 Grand Cross cheat sheet in FF14. It displays a small semi-transparent window on your desktop.

FF14を仮想フルスクリーンまたはボーダーレスで起動し、その上に常時最前面で重ねて使う想定です。ウィンドウはフォーカスを取りにくい設定にしているため、クリックしても基本的にFF14側をアクティブにしたまま操作できます。

It is intended to be used with FF14 running in borderless/windowed fullscreen mode. The overlay stays always on top and is configured to avoid taking focus, so clicking it should usually keep FF14 active.

## 言語 / Language

このオーバーレイは日本語と英語に対応しています。

- 上部の `JP / EN` ボタンで表示言語を切り替えできます。
- 選択した言語は `localStorage` に保存されます。
- ブラウザ表示などで `?lang=en` を付けると英語で開けます。

This overlay supports Japanese and English.

- You can switch languages with the `JP / EN` button in the top bar.
- The selected language is saved in `localStorage`.
- When opened in a browser-like environment, adding `?lang=en` opens it in English.

## 使い方 / How to use

### ポータブルexeで使う / Use the portable exe

GitHub Actionsで作成した `P4-grandcross-overlay-portable.exe` を起動します。

インストールは不要です。exeをダブルクリックするだけで起動します。

Run `P4-grandcross-overlay-portable.exe` generated by GitHub Actions.

No installation is required. Double-click the exe to start the overlay.

### 開発環境で試す / Run in a development environment

Node.jsを入れている場合は、次のコマンドで起動できます。

If you have Node.js installed, you can run it with the following commands.

```bash
cd p4-grandcross-overlay
npm install
npm start
```

## exeの作り方 / How to build the exe

### GitHub Actionsで作る / Build with GitHub Actions

このフォルダの中身をGitHubリポジトリ直下にアップロードすると、GitHub ActionsでWindows用ポータブルexeを作れます。

Upload the contents of this folder to the root of a GitHub repository, then GitHub Actions can build a portable Windows exe.

日本語:

1. GitHubでリポジトリを開く
2. `Actions` タブを開く
3. `Build portable exe` を選ぶ
4. `Run workflow` を押す
5. 完了後、実行結果ページ下部の `Artifacts` から `P4-grandcross-overlay-portable` をダウンロードする
6. ダウンロードしたzipを展開し、中の `P4-grandcross-overlay-portable.exe` を起動する

English:

1. Open the repository on GitHub
2. Open the `Actions` tab
3. Select `Build portable exe`
4. Click `Run workflow`
5. After it finishes, download `P4-grandcross-overlay-portable` from `Artifacts` at the bottom of the workflow run page
6. Extract the downloaded zip and run `P4-grandcross-overlay-portable.exe`

### 手元のPCで作る / Build locally

Windowsで `make-portable.bat` を実行すると、`dist` フォルダにポータブルexeが作成されます。

うまくいかない場合は、GitHub Actionsで作る方が安定です。

On Windows, run `make-portable.bat` to create the portable exe in the `dist` folder.

If local build does not work, building with GitHub Actions is usually more stable.

このアプリはフォーカスを取らないため、キー操作はElectronのグローバルショートカットで受け取ります。

| 薄くする / Decrease opacity | `Ctrl + Alt + ↓` |

| タブ右 / Tab right | `Ctrl + Alt + Shift + 1` |
| タブ左 / Tab left | `Ctrl + Alt + Shift + 2` |
| 選択枠を上へ / Move selection up | `Ctrl + Alt + Shift + 3` |
| 選択枠を下へ / Move selection down | `Ctrl + Alt + Shift + 4` |

| 選択枠を右へ / Move selection right | `Ctrl + Alt + Shift + 6` |

## パッド割り当て例 / Gamepad mapping example

Xboxアクセサリー、Steam Input、JoyToKeyなどで、パッド入力を上記のキーボードショートカットに変換して使います。

おすすめの割り当ては次の通りです。

Use Xbox Accessories, Steam Input, JoyToKey, or similar software to convert gamepad inputs into the keyboard shortcuts above.

Recommended mapping:

| パッド操作 / Gamepad input | 割り当てるキー / Key to assign | アプリ側の動作 / Overlay action |
| --- | --- | --- |
| `LB + Y` | `Ctrl + Alt + Shift + 1` | タブ右 / Tab right |
| `LB + X` | `Ctrl + Alt + Shift + 2` | タブ左 / Tab left |
| `LB + 十字上` / `LB + D-pad Up` | `Ctrl + Alt + Shift + 3` | 選択枠を上へ / Move selection up |
| `LB + 十字下` / `LB + D-pad Down` | `Ctrl + Alt + Shift + 4` | 選択枠を下へ / Move selection down |
| `LB + 十字左` / `LB + D-pad Left` | `Ctrl + Alt + Shift + 5` | 選択枠を左へ / Move selection left |
| `LB + 十字右` / `LB + D-pad Right` | `Ctrl + Alt + Shift + 6` | 選択枠を右へ / Move selection right |
| `LB + B` | `Ctrl + Alt + Shift + 7` | 決定 / Confirm |
| `LB + A` | `Ctrl + Alt + Shift + 8` | 結果タブへ / 元のタブへ戻る / Go to Result tab / Return to previous tab |
| `LB + Menu` | `Ctrl + Alt + Shift + 9` | リセット / Reset |

初期状態とリセット後は、各タブの左上ボタンに選択枠が出ます。決定後は、押したボタンの位置に選択枠が残ります。

In the initial state and after reset, the selection frame appears on the top-left button of each tab. After confirming, the selection frame stays on the button you pressed.

## 注意 / Notes

- FF14は排他フルスクリーンではなく、仮想フルスクリーンまたはボーダーレス推奨です。
- 他のアプリが同じショートカットを使っている場合、反応しないことがあります。
- パッドから直接このアプリを操作しているわけではなく、パッド入力をキーボードショートカットに変換して操作します。
- 環境によってはクリック時に一瞬フォーカスが移る可能性があります。

- Use FF14 in windowed fullscreen/borderless mode, not exclusive fullscreen.
- If another app uses the same shortcut, the overlay may not respond.
- The gamepad does not control the overlay directly. Gamepad inputs are converted into keyboard shortcuts.
- Depending on your environment, focus may briefly move when clicking the overlay.
